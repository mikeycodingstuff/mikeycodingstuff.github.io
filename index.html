Collector's App

An app built in PHP that displays a collection of albums from a database.

For each album I have displayed the name of the album, the artist name, the number of tracks and the album length.

Additional functionality was added in the ability to add a new album to the collection, and to delete any album from the collection.

Built using: PHP, MySQL

Film Quiz App

A film quiz game built in JavaScript, where the player is shown a quote from a film and has to pick the film the quote is from.

Once the game starts the player has 30 seconds to attempt to guess as many quotes as they can.

A score counter displays the correct guesses with a visual indicator when they reach a multiple of 5.

A hint button shows the correct film's release year when clicked.

Built using: JS

React Email Client

An email client built in React using supplied designs and an API with email data.

Displays the user's inbox as well as sent emails.

Email list shows a preview of all emails. The user can select a preview to open the complete email.

Built using: React, SCSS

OOP PHP listing app

An object-oriented PHP project, styled with bootstrap, that fetches and displays a list of biscuits from a MySQL database.

Each biscuit shows a total dunk and flunk count and selecting a biscuit shows more information.

Users can 'dunk' or 'flunk' biscuits, and view their favourite (dunked) biscuits in a separate page.

A hydrator is used to populate data which is displayed with a viewhelper.

Custom exception handling was used to deal with errors.

Built using: PHP, MySQL
